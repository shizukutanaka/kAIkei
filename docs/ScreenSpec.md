# kAIkei — 画面仕様書

> 本ドキュメントはkAIkeiの主要画面のワイヤーフレーム・入力制御・検索条件・アクセシビリティ要件を定義する。
> フロントエンド実装（React + TailwindCSS + shadcn/ui）の基準資料である。

---

## 1. 共通画面仕様

### 1.1 レイアウト構成

```
┌─────────────────────────────────────────────────────┐
│  TopBar: ロゴ │ 会社セレクタ │ 通知ベル │ ユーザーメニュー  │
├──────────┬──────────────────────────────────────────┤
│          │                                          │
│  Side    │  Main Content Area                       │
│  Nav     │                                          │
│          │                                          │
│  - ダッシュ│                                          │
│    ボード │                                          │
│  - 仕訳   │                                          │
│  - マスタ │                                          │
│  - 支払   │                                          │
│  - 給与   │                                          │
│  - 固定資産│                                          │
│  - 帳票   │                                          │
│  - 設定   │                                          │
│          │                                          │
└──────────┴──────────────────────────────────────────┘
```

### 1.2 共通ナビゲーション

| メニュー | サブメニュー | アクセス権限 |
|---------|------------|-------------|
| ダッシュボード | — | 全ロール |
| 仕訳 | 仕訳入力 / 仕訳一覧 / 承認待ち / AI推論ログ | `journal_entry`, `journal_approve` |
| マスタ | 勘定科目 / 補助科目 / 部門 / プロジェクト / 取引先 / 税区分 | `master_manage` |
| 支払 | 支払申請 / 全銀出力 / 銀行消込 / 資金繰り | `payment_manage` |
| 給与 | 給与計算 / 社員マスタ / 扶養控除 / e-Gov送信 | `payroll_manage` |
| 固定資産 | 資産一覧 / 償却実行 / 除却・売却 | `asset_manage` |
| 帳票 | 試算表 / 総勘定元帳 / 損益計算書 / 貸借対照表 / 監査パッケージ出力 | `report_view` |
| 設定 | 会社情報 / ユーザー・ロール / 承認ポリシー / セキュリティ / ライセンス / Webhook | `admin` |

### 1.3 共通UIパターン

| パターン | 仕様 |
|---------|------|
| 一覧画面 | DataTable（shadcn/ui Table）+ ページネーション + ソート + フィルタ |
| 詳細画面 | モーダルダイアログ or サイドシート（右側スライドイン） |
| 入力フォーム | shadcn/ui Form + React Hook Form + Zodバリデーション |
| 金額入力 | カンマ区切り表示・小数点4桁まで対応・マイナス入力可 |
| 日付入力 | DatePicker（和暦/西暦切替可） |
| 検索条件 | 折りたたみ式アコーディオン（デフォルト展開） |
| ローディング | Skeleton（shadcn/ui Skeleton） |
| エラー表示 | インラインエラー（フォーム項目直下）+ トースト通知 |
| 確認ダイアログ | 破壊的操作（削除・無効化）前にConfirm Dialog表示 |
| ショートカット | `Ctrl+S`（保存）, `Ctrl+Enter`（確定）, `Esc`（キャンセル） |

### 1.4 アクセシビリティ要件

| 要件 | 仕様 |
|------|------|
| WCAG準拠 | WCAG 2.1 AA |
| キーボードナビゲーション | 全操作をキーボードのみで完了可能 |
| スクリーンリーダー | aria-label・role属性の適切な付与 |
| カラーコントラスト | 最小4.5:1（通常テキスト）、3:1（大テキスト） |
| フォーカスインジケータ | 全インタラクティブ要素にvisible focus ring |
| フォントサイズ | 最小14px（デフォルト16px推奨） |

```

│  取引日: [2026/06/26 ▼]    伝票種別: [振替 ▼]          │
│  摘要:   [                              ]              │
├─────────┬──────┬──────┬──────┬──────┬──────┬────────┤
│ 行      │ 借貸 │ 科目  │ 補助  │ 部門  │ 税区分│ 金額(税抜)│
├─────────┼──────┼──────┼──────┼──────┼──────┼────────┤
│ 1       │ 借方 │ 1110 │ 01   │ 営業  │ 課10%│ 10,000  │
│         │      │ 現金 │      │      │      │         │
├─────────┼──────┼──────┼──────┼──────┼──────┼────────┤
│ 2       │ 貸方 │ 2120 │      │      │ 課10%│ 10,000  │
│         │      │ 売上 │      │      │      │         │
├─────────┼──────┼──────┼──────┼──────┼──────┼────────┤

├─────────────────────────────────────────────────────┤

│  ⚠ 貸借不一致 (差額: ¥1,000)                         │

└─────────────────────────────────────────────────────┘
```

### 3.2 入力項目・制御

| 項目 | 型 | 必須 | 入力制御 | バリデーション |
|------|-----|------|---------|--------------|
| 取引日 | DATE | ✅ | DatePicker（和暦切替可） | 当日以降の入力は警告（未来日） |
| 伝票種別 | SELECT | ✅ | 振替/入金/出金 | — |
| 摘要 | TEXT | — | 最大200文字 | — |
| 行番号 | AUTO | ✅ | 自動採番 | — |
| 貸借区分 | SELECT | ✅ | 借方/貸方 | — |
| 勘定科目 | COMBOBOX | ✅ | 科目マスタから検索（コード/名称の部分一致） | 有効な科目コードのみ |
| 補助科目 | COMBOBOX | — | 選択した科目に紐づく補助科目のみ表示 | — |
| 部門 | SELECT | — | 部門マスタ | — |
| 税区分 | SELECT | ✅ | 課税10%(内税)/課税10%(外税)/軽減8%/不課税 | — |
| 金額(税抜) | NUMBER | ✅ | カンマ区切り・マイナス可 | ≠0 |
| 消費税額 | NUMBER(RO) | — | 税区分から自動計算（編集可） | — |
| 金額(税込) | NUMBER(RO) | — | 税抜+消費税の自動計算 | — |

### 3.3 リアルタイムバリデーション

| ルール | タイミング | 表示 |
|--------|----------|------|
| VAL-001: 貸借一致 | 行追加/金額変更時 | 借方・貸方合計の差額を赤字表示 |
| VAL-002: 必須項目 | フォーカスアウト時 | 該当項目の下に赤字エラー |
| VAL-003: 科目有効性 | 科目選択時 | 無効科目の場合は警告 |
| VAL-004: 税区分整合 | 税区分選択時 | 科目の税区分デフォルトとの差異を警告 |
| VAL-005: 金額範囲 | 金額入力時 | 0円の場合はエラー |

### 4.2 検索条件

| 条件 | 型 | 演算子 | APIパラメータ |
|------|-----|-------|-------------|
| 取引日(From) | DATE | >= | `transaction_date_from` |
| 取引日(To) | DATE | <= | `transaction_date_to` |
| 承認状態 | SELECT | = | `approval_status` |
| 勘定科目 | TEXT | = | `account_code` |
| キーワード | TEXT | LIKE | `summary` (部分一致) |
| 起票者 | SELECT | = | `created_by` |

| 承認状態 | ✅ | 100px | バッジ色: draft=灰, waiting=黄, approved=緑, posted=青, void=赤 |
